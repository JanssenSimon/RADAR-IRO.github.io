Simulation gravitationnelle

Cette activité consiste à créer une simulation gravitationnelle numérique à n corps. Pour ce faire nous implémenterons l'algorithme décrit dans le document ci-dessous, An Exactly Conservative Integrator for the n-Body Problem.

Document mentionné: An Exactly Conservative Integrator for the n-Body Problem par Oksana Kotovych et John Bowman

L'algorithme décrite dans le paper de 2002 fait référence à cet article-ci, qui est importante pour l'implémentation: Exactly conservative integrators
